Double Solenoid (PCM)

What is it?

A Double acting solenoid that is plugged into a Pneumatics Control Module (PCM). It controls a pneumatic cylinder to create linear motion.

Properties

Forward PCM (Solenoid)
The PCM CAN ID that this solenoid is plugged into.
Forward Port (Solenoid)
The port on the PCM that this solenoid output is plugged into.
Reverse PCM (Solenoid)
The PCM CAN ID that this solenoid is plugged into. Note: This must match the Forward PCM
Reverse Port (Solenoid)
The port on the PCM that this solenoid output is plugged into.
